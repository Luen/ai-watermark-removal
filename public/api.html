POST /remove-watermark

Remove watermark from an image using AI

Request Parameters

image File Required
The image file to process. Supported formats: PNG, JPG, JPEG

Response

Returns a JSON object containing the processing results and the processed image.

Success Response

{ "success": true, "text": "AI Response text", "image": "base64 encoded image data", "watermarkRemoved": true, "hasWatermark": true, "explanation": "Detailed explanation of the watermark detection and removal" }

Error Response

{ "success": false, "error": "Error message describing what went wrong" }

POST /detect-watermark

Detect if an image contains a watermark

Request Parameters

image File Required
The image file to analyze. Supported formats: PNG, JPG, JPEG

Response

Returns a JSON object containing the watermark detection results.

Success Response

{ "success": true, "hasWatermark": true, "explanation": "Detailed explanation of what watermark was detected" }

Error Response

{ "success": false, "error": "Error message describing what went wrong" }

GET /health

Check the health status of the API

Response

Returns a JSON object indicating the API status.

Success Response

{ "status": "ok" }
